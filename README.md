## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Allow requests from Vue.js front-end ###

```sh
npm install cors
```

### Install body parser ###

```sh
npm install @types/body-parser
```

### Test the components ###

```sh
npm i --save-dev @types/jest
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
